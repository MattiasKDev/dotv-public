# statsvault
- Track player statistics including levels, XP, damage, and raid counts
- Click the profile icon (top left) to toggle the UI
- [download](https://github.com/MattiasKDev/dotv-public/raw/refs/heads/main/userscripts/statvault.user.js) (.user.js file, Tampermonkey should recognize and install)

## Accessing stored data
1. Open the Tampermonkey dashboard
2. Go to **Settings** → set **Config mode = Advanced**
3. While in dotv, click the Tampermonkey icon
4. Click the arrow next to **stat vault** → **Edit**
5. Open the **Storage** tab

> Note: The Storage tab is only visible in Advanced mode
